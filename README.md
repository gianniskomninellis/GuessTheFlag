# 🌍 Flag Guesser

A simple and interactive flag guessing game built with Python and the Turtle graphics library.

The game displays a randomly generated national flag, and the player must correctly identify the country. Each correct answer earns a point, while incorrect answers reduce the player's remaining lives. The game continues until all flags have been guessed or the player runs out of lives.

---

## ✨ Features

- Interactive graphical interface using Python Turtle.
- Random flag selection for every round.
- Score tracking system.
- Lives system to increase the challenge.
- Multiple country flags included:
  - Germany
  - Bangladesh
  - Netherlands
  - Japan
- Prevents correctly guessed flags from appearing again.
- Lightweight and beginner-friendly project.

---

## 🛠 Requirements

To run this project, you need:

- Python 3.x
- Turtle module (included with the standard Python installation)
- Random module (included with Python)

No additional libraries are required.



## 🎮 How to Play

1. Run the game.
2. A flag will appear in the Turtle graphics window.
3. Enter your guess in the console.
4. Earn points for correct answers.
5. Lose a life for each incorrect answer.
6. The game ends when:
   - All flags have been guessed, or
   - You run out of lives.
